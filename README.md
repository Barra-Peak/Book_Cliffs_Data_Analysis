# Book_Cliffs_Data_Analysis
R markdown files used to compile data from Pujols et al. (2020, 2021) with new Raman data, perform data analysis, and generate plots. Aesthetic improvements to plots were done in Adobe Illustrator.

This repository holds the compiled U-Pb and (U-Th)/He thermochronology data published in Pujols et al. (2020, 2021) in a machine-readable format as used in combination with new Raman spectroscopy data in Peak et al. (In Review, Basin Research). R Markdown files used for data analysis and visualization are also included. Note that many of the figures generated in R markdown were subsequently altered in Adobe Illustrator for aesthetic and legibility purposes (changed font size, color schemes, etc.).

[![DOI](https://zenodo.org/badge/1218545343.svg)](https://doi.org/10.5281/zenodo.20298142)

Files contained in this repository:
1. upb_zhe_data.xlsx - Detrital zircon U-Pb and (U-Th)/He data compiled from Pujols et al. (2020; 2021) in a machine-reader format
2. 2025_04_10_alpha_dose_data.xlsx - Raman radiation damage data
3. models_grenville_cooling.xlsx - forward model time-temperature points for main models discussed in Peak et al. (In Review)
4. models_grenville_cooling_w_forland.xlsx - forward model time-temperature points for alternative model with Grenville foreland burial
5. models_slow_early_cooling.xlsx - forward model time-temperature points for alternative model with slow initial cooling
6. models_fast_late_cooling.xlsx - forward model time-temperature points for alternative model with fast initial cooling
7. model_predictions.xlsx - HeFTy forward model ZHe date and radiation damage predictions
8. Book_Cliffs_Figures.RMD - R Markdown file used to make data plots and other plot-type figures
